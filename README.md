# pdf_page_counter

Count the number of pages in a PDF file.

## HOW TO USE

```
> go run main.go
```

Access http://localhost:8/ with Chrome or FireFox.

## REQUIRES

- Windows
- Go (1.9 or above)
- nkf
- pdfinfo

\* Put nkf.exe and pdfinfo.exe in the same directory as main.go.
